# ITalker
慕聊-MVP模式编写的带服务端的IM即时通讯APP  
慕课网《带服务端的IM即时通讯APP实战》学习成果  
由于这个App需要配置服务端，所以放安装包就是没有意义的了  
导入代码后，需要在Commom包下面修改baseUrl为你的本地url即可
